Memory Cards

Game Development

Games development is another service that I offer so I wanted to include an example of a game on my portfolio section.

A simple game, but one that would fit nicely into a website or web application. The aim of the game is to match all the pairs in the fastest time possible.

One of the biggest challenges I was faced with whilst building this game was how to keep track and compare the users two card selections. I had to come up with a way to store the users first selection, compare it to the second selection and if there is a match, fade the cards out.

The application is fully responsive and works on all devices. Give it a go, see how quickly you can match all the cards.
