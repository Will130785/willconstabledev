OUT-N-IN

Website Design and Development

This is a modern, fully responsive website, designed for a fast food burger restaurant.

Fully designed by myself, the site features a sleek, modern user interface, a menu and shopping cart section and a fully functional contact form.

I took inspiration from the major fast food burger joints out there but added my own touch to give it an original feel. It also features a gallery section which was built using CSS grid.

The final product resulted in a website perfect for a modern day restaurant.
