Website Design and Development

This is my personal portfolio site and I wanted to include this as it contains some nice features that could potentially be encorporated into your own website.

The site features a parallax scrolling and custom animations to give a smooth overall user experience.

As an extra I also included a section where the user can type in any city in the world and have the current weather displayed on screen. This was achieved by connecting to the weather API to get the data for the corresponding city.

The site also contains a mini game, where the user is required to match the RGB colour value to one of the coloured circles on the screen. If the user guesses right, the background changes to that colour.
