Blackhawk Oven Cleaning & Property Services

Customer Booking System

The first part of a double collaboration with Blackhawk Oven Cleaning & Property Services. I worked with the client to develop a customer database and booking system.

A web based system that allows the user to store client details and also book jobs. The booking system sends an SMS message to the customer with the details of the job. The day before a booking, a reminder email automatically gets sent to the technician assigned to the job. The booking also gets added to a reminder section where all the jobs for the next day can be viewd.

When a customer is entered into the database they are sent a text advising them of this and to check their details are correct. The system also sends an email every six months reminding Blackhawk staff to reconnect with the client.

The biggest challenge involved with this project was developing the logic behind triggering the reminder emails.
